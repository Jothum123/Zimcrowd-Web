### Method 2: Smartphone Camera (GOOD)

**Use when scanner not available**

#### Setup:
1. **Find good lighting**
   - Natural daylight (near window) is best
   - Bright LED light works too
   - Avoid direct sunlight (causes glare)
   - Avoid yellow/warm lights (use white light)

2. **Prepare surface**
   - Use a dark, non-reflective surface
   - Place ID flat on surface
   - No patterns or text underneath

3. **Camera position**
   - Hold phone directly above ID (not at angle)
   - Distance: 20-30cm (8-12 inches)
   - ID should fill 60-80% of frame
   - Keep phone parallel to ID

#### Camera Settings:
- **Focus:** Tap on ID to focus
- **Flash:** OFF (causes glare)
- **HDR:** OFF (can blur text)
- **Resolution:** Highest available
- **Zoom:** Don't use digital zoom (get closer instead)

#### Taking the Photo:
1. Hold phone steady (use both hands)
2. Take a deep breath
3. Tap shutter button gently
4. Take 2-3 photos (choose best one)
5. Review photo before uploading:
   - Zoom in to check text is sharp
   - All text is readable
   - No blur or motion

**Expected Result:**
- âœ… Confidence: 70-85%
- âœ… Good text extraction
- âœ… Most fields parsed correctly

## âŒ Common Mistakes to Avoid

### 1. **Blurry Images**
**Problem:** Camera moved during capture
**Solution:** 
- Hold phone steady with both hands
- Lean against wall for stability
- Use timer mode (2-3 second delay)
- Use tripod or phone stand

### 2. **Poor Lighting**
**Problem:** Dark shadows, yellow tint, glare
**Solution:**
- Use natural daylight
- Add more light sources
- Position lights at 45Â° angles
- Avoid direct overhead lights

### 3. **Glare/Reflections**
**Problem:** Shiny ID surface reflects light
**Solution:**
- Turn off flash
- Angle ID slightly (5-10Â°)
- Use diffused light (through curtain)
- Avoid direct light sources

### 4. **Low Resolution**
**Problem:** Image too small, pixelated
**Solution:**
- Use highest camera resolution
- Get closer to ID (don't zoom)
- Don't crop too much
- Save as PNG or high-quality JPG

### 5. **Wrong Angle**
**Problem:** ID appears distorted, text slanted
**Solution:**
- Hold phone directly above ID
- Keep phone parallel to ID
- Use grid lines to align
- Check edges are straight

### 6. **Partial ID**
**Problem:** Corners or edges cut off
**Solution:**
- Include full ID in frame
- Leave small margin around edges
- Check preview before capturing

### 7. **Dirty/Damaged ID**
**Problem:** Scratches, stains, worn text
**Solution:**
- Clean ID gently with soft cloth
- If text is worn, may need to reapply for new ID
- Capture in best possible condition
